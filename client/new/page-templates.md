# Jurri Template - Page Templates Reference

## Homepage Structure

### 1. Navigation Header
- Fixed position, dark background
- Logo on left
- Menu items centered
- CTA button on right
- Hamburger menu on mobile

### 2. Hero Section
- Full-width dark navy background
- Large serif headline (Frank Ruhl Libre)
- Subheadline in lighter weight
- Gold CTA button
- Optional background overlay/image

### 3. Services Overview
- Grid of service cards (3-4 columns on desktop)
- Icon or visual for each service
- Title + brief description
- Hover animations
- Link to detailed service page

### 4. About/Value Proposition
- Split layout: image + text
- Statistics/numbers with labels
- Core values or differentiators
- Gold accent elements

### 5. Testimonials Section
- Carousel or grid layout
- Quote text in italics
- Client name and title
- Company logo
- Rating stars (optional)

### 6. CTA Banner
- Full-width dark/gold section
- Compelling headline
- Action button
- Contact information

### 7. Footer
- Multi-column layout
- Company info column
- Service links column
- Contact info column
- Social media icons
- Copyright

## About Page Structure

### 1. Hero Banner
- Page title in serif
- Breadcrumb navigation
- Background image/gradient

### 2. Company Story
- Two-column layout
- Narrative text
- Key statistics
- Timeline (optional)

### 3. Core Values
- Grid of value cards
- Icon + title + description
- Consistent styling

### 4. Leadership Team
- Profile cards
- Headshot images
- Name, title, credentials
- LinkedIn links
- Brief bios

### 5. Awards/Recognition
- Logo carousel
- Certification badges
- Industry affiliations

## Services Page Structure

### 1. Hero Section
- Service category title
- Brief description
- CTA button

### 2. Service Details
- Detailed description
- Benefits list
- Process/methodology
- Supporting visuals

### 3. Sub-services Grid
- Cards for each sub-service
- Icon/visual
- Title
- Brief description
- Learn more link

### 4. Case Studies/Results
- Statistics
- Client success stories
- Before/after metrics

### 5. Related Articles
- Blog post cards
- Thumbnail, title, excerpt
- Read more link

## Contact Page Structure

### 1. Contact Header
- Page title
- Brief intro text

### 2. Contact Form
- Full name
- Email
- Phone
- Company
- Message
- Submit button

### 3. Contact Information
- Address
- Phone number
- Email
- Business hours

### 4. Map (Optional)
- Embedded map
- Location marker

## Blog/Articles Structure

### 1. Articles Grid
- Featured article (large)
- Recent articles (grid)
- Category filters
- Search functionality

### 2. Article Card
- Thumbnail image
- Category tag
- Title
- Excerpt
- Author
- Date
- Read time

### 3. Article Page
- Hero image
- Title
- Meta (date, author, category)
- Body content
- Related articles
- Share buttons
